Feedback Collection

The Feedback Collection feature allows you to gather user feedback when they interact with the chatbot.

You can enable feedback collection based on how you're using the chatbot:

  • Shortcode
  • Widget
  • Custom Bots
  • Go to the Chat Bot - Shortcode tab in your dashboard.
  • Navigate to the Interface tab.
  • Here, you can enable the feedback collection feature.
  • Go to the Chat Bot - Widget tab in your dashboard.
  • Navigate to the Interface tab.
  • Here, you can enable the feedback collection feature.
  • Go to the Chat Bot - Custom Bots tab in your dashboard.
  • Select your bot and hit the Edit button.
  • Navigate to the Interface tab.
  • Here, you can enable the feedback collection feature.
info

When feedback collection is enabled, logging is automatically activated. This means the system will save logs of conversations to store the feedback provided by users.

Customizing Feedback UI​

In the same Interface tab, you can also customize or translate the text displayed in the feedback UI:

  1. Feedback Title: The title displayed in the feedback window. The default is "Feedback."
  2. Feedback Message: A short description below the title. The default is "Please provide details: (optional)."
  3. Feedback Success Message: The message shown after feedback is submitted. The default is "Thank you for your feedback!"

Once feedback collection is enabled, two icons (thumbs up and thumbs down) will appear under each AI response. Users can click these icons to provide feedback. The feedback window inherits its style (colors, fonts, etc.) from the chat window for a consistent appearance.

Viewing User Feedback​

You can view the feedback provided by users by following these steps:

  1. Navigate to the Chat Bot - Logs tab.
  2. In the logs table, you will see a column labeled Feedback for each conversation. This column shows whether feedback was provided and a short version of the user's comments if any.
  3. Each conversation may have multiple feedback entries, but the latest one will be displayed in the table.
  4. To view all feedback and the entire conversation, click the View conversation icon. This will open the full conversation where you can see all feedback, including the full version of user comments.

This setup helps you monitor and analyze how users are interacting with your chatbot and how they respond to its output.
