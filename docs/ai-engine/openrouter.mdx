## Bring Your Own Key Model

Our plugin uses a *Bring Your Own Key* model, acting as a bridge between your WordPress site and OpenRouter AI services.

You need a OpenRouter account and API credits from OpenRouter to use our plugin. This approach lets you control your API keys and ensures access to the latest OpenRouter AI models.

:::info
Buying our plugin does not include OpenRouter credits. You need to purchase credits separately from OpenRouter. Check their [pricing page](https://openrouter.ai/credits) for details.
:::

## Generate OpenRouter API Key

Generating an OpenRouter API key is straightforward. Follow these steps:

1. Open your web browser and go to [OpenRouter home page](https://openrouter.ai/). Create an account and sign in.
2. Now go to [OpenRouter API Key page](https://openrouter.ai/keys).
3. Click "Create key".

<img src={createkey} />

4. Give your api key a name and hit "Create".

<img src={createkey2} />

5. Copy the API key and save it in a secure location. You will need it to integrate OpenRouter services into our plugin.

<img src={createkey3} />

## Setting Up AI Power Plugin

Set up the AI Power plugin to use OpenRouter services:

- Go to the AI Power plugin on your WordPress dashboard.
- Go to the **Dashboard** tab.
- Select **OpenRouter** from the provider dropdown list.
- Enter your OpenRouter API key.

<img src={openroutersettings} />

Once api key is saved, now you can sync latest models. Hit Sync button to get the latest models.
